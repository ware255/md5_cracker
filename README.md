# md5_cracker
md5を辞書攻撃するツールです。総当たり攻撃の実装めんどいんでやめました。気が向いたら実装するかも？

## インストール
```
$ sudo apt install git g++ make
$ git clone https://github.com/ware255/md5_cracker.git
$ cd md5_cracker
$ make
```

## 実行方法
```
$ md5_cracker <hash> <file_name>
```
